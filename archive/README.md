# Archived Notebooks

These notebooks cannot be run without unavailable dependencies (private datasets, API keys, or internal packages). They are preserved here for reference.

## Contents

- **intro-finance.ipynb** — Basic finance concepts (stocks, ETFs, options). Requires `TIINGO_API_KEY` and private HDF5/CSV data.
- **exploring_visualizations.ipynb** — Options visualisations. Imports a private `backtester` package.
- **hdf5_store.ipynb** — Data engineering notebook for converting private SPX CSV data to HDF5.
- **dask_dataframes.ipynb** — Dask wrappers for working with private HDF5 options data.
