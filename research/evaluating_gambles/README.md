# Evaluating Gambles

An exploration of expected values, time averages, and optimal betting strategies.

Covers the St. Petersburg paradox, utility functions, non-ergodic multiplicative dynamics, and the Kelly criterion for optimal bet sizing.

## References

- [Evaluating gambles using dynamics](https://doi.org/10.1063/1.4940236) — Peters & Gell-Mann
- [Ergodicity Economics lecture notes](https://ergodicityeconomics.com/lecture-notes/) — Peters

## Run

```bash
python run.py
```

Self-contained, no external data needed.
